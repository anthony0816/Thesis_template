\chapter*{CONCLUSIONES GENERALES}
\addcontentsline{toc}{chapter}{CONCLUSIONES GENERALES}

\begin{enumerate}
    \item La fundamentación teórica realizada permitió identificar los vacíos existentes en las herramientas actuales de gestión de activos digitales y seleccionar un \textit{stack} tecnológico basado en tecnologías web modernas y metodología ágil SCRUM.

    \item Se implementó una aplicación web funcional que integra creación, almacenamiento, localización y reutilización de recursos digitales, cumpliendo con los requisitos definidos y constituyendo una herramienta de apoyo real para desarrolladores de videojuegos.

    \item La validación mediante pruebas de funcionalidad, integración y usabilidad demostró que el sistema es fiable, eficiente y valorado positivamente por los usuarios, con un índice de usabilidad SUS de 82 puntos.
\end{enumerate}
